 4. Создать внешний репозиторий c названием JSON.-------- Repositories -> JSON -> create
 5. Клонировать репозиторий JSON на локальный компьютер. -------- git clone https://github.com/hopesinn/JSON.git
 6. Внутри локального JSON создать файл “new.json”.-------- cd JSON -> touch new.json
 7. Добавить файл под гит.-------- git add new.json
 8. Закоммитить файл.-------- git commit -m "first commit"
 9. Отправить файл на внешний GitHub репозиторий.-------- git push
 10. Отредактировать содержание файла “new.json” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате JSON. 
-------- vim new.json -> I ->

        "FirstName":"Hope",
        "Age":"20"
}

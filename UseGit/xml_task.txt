XML
 21. Создать внешний репозиторий c названием XML.-------- Repositories -> XML -> create
 22. Клонировать репозиторий XML на локальный компьютер.-------- git init -> git clone https://github.com/hopesinn/XML.git
 23. Внутри локального XML создать файл “new.xml”.-------- cd XML -> touch new.xml
 24. Добавить файл под гит.-------- git add new.xml
 25. Закоммитить файл.-------- git commit -m "commit"
 26. Отправить файл на внешний GitHub репозиторий.-------- git push
 27. Отредактировать содержание файла “new.xml” - написать информацию о себе (ФИО, возраст, количество домашних животных, будущая желаемая зарплата). Всё написать в формате XML. 
-------- vim new.xml -> I -> 

 <new.xml>
        <FirstName>Hope</FirstName>
        <Age>20</Age>
</new.xml>


esc -> : -> wq

 28. Отправить изменения на внешний репозиторий.-------- git add new.xml -> git commit -m"commit" ->  git push
 29. Создать файл preferences.xml -------- touch preferences.xml
 30. В файл preferences.xml добавить информацию о своих предпочтениях (Любимый фильм, любимый сериал, любимая еда, любимое время года, сторона которую хотели бы посетить) в формате XML.
-------- vim preferences.xml -> I ->

<preferences.xml>
<serial>Magnificent Century</serial>
<book>
        <author>Oscar Wilde</author>
        <name>The Picture of Dorian Gray</name>
</book>
<dish>cheesecake</dish>
</preferences.xml>


esc -> : -> wq
